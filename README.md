# Socket.io
Simple Socket.io chat using express on NodeJS

#### Description
Runs a server that pushes messages to the clients through events
- Server: `index.js`
- Client: `index.html`
- Style: `public/css/style.css`

#### How to run
- execute `start.bat`
- point your browser to `http://localhost:8080`
